# Knowledge Compilers

A knowledge compiler turns rough notes into reusable knowledge assets.

## Why this note exists

- capture is easy, but later synthesis is hard
- search helps retrieval, but not governance
- a compiler pipeline can normalize, connect, and publish notes

## Working definition

A knowledge compiler is a system that reads raw notes, extracts structure, and materializes a more stable knowledge layer.

## Typical pipeline

1. ingest raw notes
2. extract metadata and topic candidates
3. reconcile overlapping topics
4. build concept and source pages
5. publish selected content into a public layer

## Comparison

A search-only note archive stores text well, but a compiler-oriented workflow also tries to build explicit topic structure and reusable pages.
